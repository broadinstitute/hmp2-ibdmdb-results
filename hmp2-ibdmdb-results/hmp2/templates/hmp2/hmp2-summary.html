Results

This page shows the high level results over all of the HMP2 pipelines. Each run is comprised of a set of data that has been uploaded to the HMP2 servers. Once there, it is filtered for quality and error checked for completeness and saved under the raw files page. After the QC phase, the data is run on a specific Anadama pipeline, producing several types of data products. Each data product for a project is saved on the products.
