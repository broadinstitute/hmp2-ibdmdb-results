File Descriptions

Peptide Roll-up

Peptide identification files created by the MSGF+ software (v10072) using the HMP 1 gut reference genomes. Sets of files generated using both 1% and 5% False Discovery Rate (FDR) and one or two unique peptide per protein identification. Files provided in both TSV and BIOM formats.

Merged Files

All peptide roll-up files at the one unique peptide per protein and 1% FDR level were summarized at both the EC and KO level and merged into TSV and BIOM file formats for dissemination.
