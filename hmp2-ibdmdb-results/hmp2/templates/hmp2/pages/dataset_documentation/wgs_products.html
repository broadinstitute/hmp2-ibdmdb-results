File Descriptions

Gene Calls

Archive contain gene calls in GFF3 format and protein coding sequences in FASTA format. Gene calls were predicted using the IGS Metagenomics Pipeline.

Contigs

Assembled contigs produced using the IGS Metagenomics Pipeline

Functional Profiles

Archive contains predicted genefamilies, ECs, pathway abundances and accompanying log file all generated using the HUMAnN2 software package.

Taxonomic Profiles

Abundance tables as generated by MetaPhlAn2 in TSV, BIOM and merged table configurations.
